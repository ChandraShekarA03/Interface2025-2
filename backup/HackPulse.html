Technical Event

HackPulse CTF

HackPulse CTF is a thrilling, offline cybersecurity competition built around the Capture The Flag (CTF) format. Participants will face real-time challenges across multiple domains, including:

  • Cryptography
  • Web Exploitation
  • Reverse Engineering
  • Digital Forensics

The goal: Capture digital flags hidden in each problem by breaking down vulnerabilities, decrypting secrets, and analyzing code.

Event Format

Offline CTF Event Plan using CTFd. We will host a fully offline Capture-The-Flag competition using CTFd for 60 teams (120 students) by leveraging three laptops and a TP-Link router to create an isolated network.

Setup Distribution:

  • Laptop 1: Hosts CTFd application and Docker containers for challenges
  • Laptop 2: Runs database (PostgreSQL with pgAdmin) and Redis for data management
  • Extra Laptop: Provides monitoring, backup support, and auxiliary tasks
  • TP-Link Router: Creates local Wi-Fi network for all 60 systems to connect simultaneously.

Team Structure

  • Team Size: 1–2 participants (MAX: 120 students)
  • Event Duration: 1.5 hours (90 minutes)
  • Organizing Team: 2 Event Heads + 6 Volunteers

Rules and Regulations

  • Teams must solve challenges to capture flags.
  • Challenges vary in difficulty: beginner to advanced.
  • Internet access is strictly prohibited; only provided resources may be used.
  • External storage devices or tools are not permitted unless explicitly provided.
  • Any form of cheating or flag sharing leads to immediate disqualification.
  • Organizers’ decisions are final and binding.

Judging Criteria

1. Point-Based Evaluation:

  • Each challenge (flag) carries specific points based on difficulty.
  • Teams accumulate points by successfully solving and submitting flags via CTFd.

2. Time-Based Tie-Breaker:

  • In case of a points tie, the team who reached the tied score first will be ranked higher.
