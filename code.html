Technical Event

Code Quest

Ultimate programming competition testing your coding skills across multiple languages and frameworks. Participants will face challenging problems that test their algorithmic thinking, problem-solving abilities, and coding efficiency.

Event Information

Duration: 3 hours

Team Size: Individual or Team of 2

Venue: Computer Lab A, Block 1

Rules & Regulations

  • Participants can use any programming language of their choice
  • Internet access will be provided for documentation purposes only
  • Code must be original and written during the competition
  • Plagiarism will result in immediate disqualification
  • Judging will be based on correctness, efficiency, and code quality

Guidelines

  • Bring your own laptop with preferred IDE/editor installed
  • Ensure your system has necessary compilers/interpreters
  • Arrive 30 minutes before the event for setup
  • Valid college ID is mandatory for participation
