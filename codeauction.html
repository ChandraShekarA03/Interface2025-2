Technical Event

Code Auction 2.0

Code Auction 2.0 is a dynamic, strategy-infused technical event where logic meets adrenaline. Teams participate in a live auction to bid on buggy code snippets and limited debugging tools. Once the gavel drops, the real challenge begins – fixing the code under strict time limits to earn maximum points.

It’s a high-stakes blend of auction tactics, code analysis, and pressure-driven problem solving. Make smart investments, identify valuable snippets, and strike the perfect balance between risk and reward to dominate the leaderboard.

Event Information

Duration: 3 hours

Team Size: 2–3 participants per team

Max Teams: 10–15 (based on time & space)

Event Structure & Flow

  • Round 1: The Debug Hunt – Solve buggy code to unlock clues. Find hidden markers and earn virtual bidding points.
  • Round 2: Auction & Access – Bid using virtual points to solve coding challenges. Code must be debugged for points.
  • Round 3: Double or Nothing – Final risk round. Score may double… or drop to zero!

Judging Criteria

  • Technical Accuracy
  • Code Efficiency
  • Auction Strategy and Budget Use
  • Debugging Speed
  • Team Coordination

Guidelines

  • Bring personal laptops with code editor installed
  • Maintain discipline and follow time slots strictly
  • Valid student ID is mandatory
  • Decisions by organizers will be final
